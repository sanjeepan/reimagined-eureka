Summary

This is the table of Computer Network Topologies summary .

Topology What it is Image
Point to Point(P2P) The network consists of a direct link between two computers
Bus Topology Uses a single cable which connects all the included nodes
Ring Topology Every device has exactly two neighboring devices for communication purpose
Star Topology All the computers connect with the help of a hub.
Mesh Topology The mesh topology has a unique network design in which each computer on the network connects to every other.
Tree Topology Tree topologies have a root node, and all other nodes are connected which forming a hierarchy.
Hybrid Topology Hybrid topology combines two or more topologies
